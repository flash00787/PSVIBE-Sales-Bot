--- IMPORT STRUCTURE ---
- app.py: Entry point. Imports bot, bot.*, bot.handlers.*
- bot/__init__.py: Core library (2100+ lines). Contains gspread helpers, API calls, config, member ops
- bot/handlers/__init__.py: Aggregates all handler modules
- Each handler: Python module in handlers/ directory

--- ERROR HANDLING PATTERN ---
- error_handler in bot/handlers (imported by app.py)
- Custom _sheets_retry decorator for gspread 429/500/503 retries
- 403 → SheetsPermissionError (no retry)
- Sheets ops wrapped in try/except with logging
