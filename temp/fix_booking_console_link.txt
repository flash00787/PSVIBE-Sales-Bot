## Architecture Analysis

### Console_Booking Sheet (gspread) — Shared Source of Truth
Columns: A=BookingID, B=Date, C=ConsoleID, D=MemberID, E=StartTime, F=EndTime, G=Status, H=Staff, I=Notes

fetch_console_status() reads this sheet. Only rows where G=Active or Scheduled mark console as busy.

### The Gap
1. **API Booking Flow** (`booking_flow.py`): Creates bookings via API (`/bookings` endpoint)
2. **Admin Approval** (`admin_bookings.py`): Approves → status=confirmed, assigns consoleId
3. **BUT**: Console_Booking sheet is NEVER updated during approval/cancellation
4. **Result**: Console still shows as "Free" in fetch_console_status() even though it's booked

### Fix Plan
1. admin_bookings.py `_do_booking_action`: After approval, write Scheduled row to Console_Booking
2. booking_flow.py `_do_cancel_booking`: After cancel, mark matching Scheduled rows as Done
3. booking.py `_do_create_booking`: Before creating Active row, clean up any Scheduled rows for same console


## Changes Applied

### 1. admin_bookings.py — Booking Confirmed → Console "Scheduled"
- Added `get_booking_sh, now_mmt` to imports
- After approval, writes a "Scheduled" row to Console_Booking sheet
- Console now shows as busy (not Free) in status board

### 2. booking_flow.py — Booking Cancelled/No-show → Console cleanup
- In `_do_cancel_booking`: After PATCH, finds and marks Scheduled row as "Done"
- In `cb_booking_arrive` (no-show): After marking no-show, cleans up Scheduled row

### 3. booking.py — Session Start → Cleanup old Scheduled rows
- Added `get_booking_sh` to imports
- Before creating Active booking row, marks any Scheduled rows for same console as "Done"
- Prevents duplicate rows causing console to remain busy after session ends

### Data Flow:
1. API Booking created (pending) → Admin approves → Console_Booking: "Scheduled" row
2. Console status board: Console shows as "Scheduled" (busy)
3. Session starts → Scheduled row → "Done", new "Active" row created
4. Session ends → Active row → "Done", console free again
5. If cancelled/no-show → Scheduled row → "Done", console free
