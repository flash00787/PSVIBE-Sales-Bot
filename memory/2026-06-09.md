### 🐛 Session End / Delete Session Duplicate Fix (08:58-09:08 UTC)

#### Bug
Session entries accumulating in `console_games` as duplicates when sessions end and restart on the same console. Boss reported "Console → Session end" is broken.

#### Root Cause
`DELETE /api/delete_session_game/{console_id}` endpoint didn't normalize space characters in `console_id`. The DB was normalized to "C-01" (no spaces) but the bot occasionally sends space-containing IDs like "C - 01". The SQL `DELETE WHERE console_id='C - 01'` returned 0 rows affected → old Session entry remained, and new one was added on top → duplicate.

#### Fix
- **API endpoint** (`app.py`): Changed to `REPLACE(console_id, ' ', '')` in the WHERE clause, matching the pattern used by `move_console_game` and `remove_console_game` endpoints
- **DB cleaned**: Deleted 2 duplicate Session entries (Fifa 2026 on C-04 ×2, C-10 ×2)

## 🐛 Account Balances Fix — GROUP BY Topup Bug (11:22-11:27 UTC)

### Symptom
Boss sent screenshot showing wrong balances (via Telegram image):
- Cash: 69,600 Ks | KPay: 292,333 Ks | Wave: 32,000 Ks | Total: 393,933 Ks
- Actually expected: Cash: 69,600 | KPay: ~382,333 | Wave: 32,000 | Total: ~483,933

### Root Cause
`patch_routes.py` topup_log query used `GROUP BY payment_method`:
```sql
SELECT payment_method, SUM(amount) as total FROM topup_log GROUP BY payment_method
```
Two topup records both had payment_method `'KPay:90000/Cash:0'` → collapsed into ONE row.
Code then parsed `pm = 'KPay:90000/Cash:0'` and extracted only **90,000 Ks** per group.
But actual total was **180,000 Ks** (2 records × 90,000).

**Lesson:** GROUP BY collapses duplicate payment_method strings, but the per-segment amount is extracted from the single string value (90,000), not the SUM total (180,000).

### Fix
Removed GROUP BY + SUM(amount), query individual rows instead:
```sql
SELECT payment_method, amount FROM topup_log WHERE topup_date >= '2026-01-01'
```
Same bug did NOT exist in dashboard_routes.py (already used individual rows, no GROUP BY).

### Files Modified
- **`/root/psvibe_api_server/patch_routes.py`** — Line 886-889: removed GROUP BY and SUM(amount) from topup_log query
