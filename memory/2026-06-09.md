### üêõ Session End / Delete Session Duplicate Fix (08:58-09:08 UTC)

#### Bug
Session entries accumulating in `console_games` as duplicates when sessions end and restart on the same console. Boss reported "Console ‚Üí Session end" is broken.

#### Root Cause
`DELETE /api/delete_session_game/{console_id}` endpoint didn't normalize space characters in `console_id`. The DB was normalized to "C-01" (no spaces) but the bot occasionally sends space-containing IDs like "C - 01". The SQL `DELETE WHERE console_id='C - 01'` returned 0 rows affected ‚Üí old Session entry remained, and new one was added on top ‚Üí duplicate.

#### Fix
- **API endpoint** (`app.py`): Changed to `REPLACE(console_id, ' ', '')` in the WHERE clause, matching the pattern used by `move_console_game` and `remove_console_game` endpoints
- **DB cleaned**: Deleted 2 duplicate Session entries (Fifa 2026 on C-04 √ó2, C-10 √ó2)
