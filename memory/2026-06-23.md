- **Root cause:** `parse_mode: "Markdown"` with HTML `<b>` tags → HTTP 400 Bad Request
- **Fix:** Changed to `parse_mode: "HTML"` consistently; converted `*bold*` → `<b>bold</b>`, `` `code` `` → `<code>code</code>`

#### Type 3: Event Loop Issue
- **Root cause:** `resume_active_timers()` ran at module load time (app.py import) when no event loop existed → timer tasks created but never ran
- **Fix:** Moved to FastAPI `lifespan` handler — runs inside uvicorn's event loop
- **Result:** `"Timer task STARTED"` logs confirmed appearing; BK#865 (C-04), BK#867 (C-10) timers running
