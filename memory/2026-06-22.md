## Key Learnings
1. **DB times are MMT** (Myanmar Time, UTC+6:30), not UTC
2. **`now_mmt()` usage**: API stores session start times as MMT
3. **Bot timer calc**: `_start.replace(tzinfo=MMT)` treats stored time as MMT → subtracts from `now_mmt()` → correct for MMT-stored times
4. **Cloudflare tunnel caching**: Even with origin no-cache headers, Cloudflare CDN may cache static assets. Need aggressive cache-busting or CDN purge.
5. **Vite hash stability**: Same source code → same chunk hashes → Cloudflare serves stale cached version even when origin is updated
