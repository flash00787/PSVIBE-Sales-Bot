### Lessons Learned
| # | Lesson |
|:-:|--------|
| 195 | **MySQL DECIMAL columns return strings in Node.js mysql2 driver** — ALWAYS wrap with `Number()` when using `+` operator. The `-` operator works correctly with strings (coerces to number), but `+` concatenates. Systematic audit required for all MySQL → API responses. |
| 196 | **Cloudflare Tunnel breaks session cookies** — Set-Cookie headers from backend are NOT forwarded by Cloudflare Tunnel for standard fetch() calls. Query-param token auth (`?token=***`) is the most reliable approach. |
| 197 | **Cache-busting URL format matters** — Always append version as query parameter AFTER the filename (`file.js?v=123`), not as a separate path segment (`file/?v=123`). |
| 198 | **Vue CDN zero-build: avoid arrow function scope issues** — `data()` returning object with arrow function using `r` can break if `r` is defined outside the arrow but references don't propagate. Use regular functions in `render`. |
