### Mobile Crash Debugging — Critical Lessons
| # | Lesson |
|:-:|--------|
| 109 | **Xray restart blocks HTTP response** — `os.system("systemctl restart xray 2>/dev/null &")` still blocks despite `&`. The shell waits for the process descriptor. Fix: `subprocess.Popen(["systemctl","restart","xray"], ...)` for truly non-blocking. (#109) |
| 110 | **Xray config save must be background thread** — Even with Popen, the `save_xray_clients()` call itself takes time. Moving it to `threading.Thread(daemon=True)` ensures HTTP response sends BEFORE config I/O. (#110) |
| 111 | **`btn.disabled=true` on onsubmit blocks mobile browsers** — Confirmed lesson #104. Mobile Safari/Chrome ignores form submission when submit button is disabled. Fix: use `data-submitted` flag + visual CSS only. (#111) |
| 112 | **Cloudflare + mobile + POST = fragile** — POST through Cloudflare to mobile networks can fail silently. Content-Length + flush + fast response (< 200ms) essential. (#112) |
