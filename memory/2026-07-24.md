### 1. Timeline Vue SPA — Data မပေါ်တဲ့ Bug (429 Rate Limit) ✅
- **Root cause:** `getTimelineBookings()` က `/api/bookings` ကို status ၇ခုအတွက် တပြိုင်နက်ခေါ်တယ်။ `getAllBookingsCount()` က နောက်ထပ် ၇ခု။ **Total: 16 parallel API calls** → Cloudflare 429 Too Many Requests → data အကုန်ပျက်
- **Fix:** 7 separate status calls → **1 single call** `/api/bookings?date=D` (no status filter, JS side `.filter()` နဲ့ group)
- **Files:**
  - `/root/psvibe-dashboard/src/api/timeline.ts` — `getTimelineBookings()` rewritten: 1 call + `.filter()`, `getAllBookingsCount()` rewritten: 1 call
  - `/root/psvibe-dashboard/src/views/TimelineView.vue` — Removed `getGames` from `fetchData` (not needed), removed debug logs

### 2. BranchStore 401 Unauthorized ✅

- **Fix:** Changed to `apiClient.get()` which auto-attaches JWT via interceptor
- **File:** `/root/psvibe-dashboard/src/stores/branchStore.ts`
