- **Root cause:** `esc()` function had `{ {` (double brace) → `function esc(s) { { return ...; }` — missing closing `}` caused `Unexpected end of input` at end of script.
- **Effect:** Entire JavaScript was syntactically invalid → NO JavaScript executed on page load → `switchTab()` never ran → all tabs hidden (empty main content). Only CSS worked (sidebar, modals).

- **Lesson #161:** Always run `node --check` after ANY HTML/JS changes — one extra `{` kills entire page.
- **Fix:** Removed extra `{` → `function esc(s) { return ...; }` ✅

## Outline VPN — Trial Key Count Root Cause + Fix ✅
- **Bug:** `get_agent_trial_key_count()` queries commissions WHERE `duration_days = 3`, but `record_commission()` was guarded by `if base_price > 0:` — Free Trial has `base_price = 0.0`, so NO commission entry was ever created for trial keys. Count always returned 0.
- **Effect:** Trial limit (`max_trial_keys`) was effectively infinite — agents could create unlimited trial keys.
- **Fix:** Removed `if base_price > 0:` guard. `record_commission()` now always called regardless of price. Trial keys create commission records with `duration_days = 3` and `commission_amount = 0`.
- **Files modified:** `/opt/outline-web/server.py` — 4 locations (Handler + AgentOnlyHandler, Outline + Xray create)
- **Lesson #163:** When filtering on commission records, ensure the flow actually creates the records. `base_price > 0` guard skipped trial keys entirely.
