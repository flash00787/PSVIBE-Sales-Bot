### Lessons Learned
- **#56: API endpoint needs AppLayout wrapper** — Vue pages rendered via router-view don't auto-inherit sidebar; must wrap in `<AppLayout>` component
- **#57: Dashboard uses JWT not x-api-key** — Dashboard authenticates via `apiClient` (axios with Bearer token), not raw `x-api-key` header
- **#58: Rebook = same phone + same date + same game → Done** — Booking cancellations that are rescheduled and completed shouldn't count as failures
- **#59: Console breakdown excludes empty console_id** — Some early bookings have no console assigned; `console_id != ''` filter causes mismatch between total and console sum
