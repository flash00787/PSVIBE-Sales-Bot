2. **`routes/finance_routes.py`** — Auto-balancer threshold: only adjusts for gaps < 50K; gaps ≥ 50K log warning and skip
3. **`patch_routes.py`** — Fixed duplicate `api_finance_balance_sheet` function (route pointed to empty one, returning null)
