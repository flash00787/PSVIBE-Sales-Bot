### 9. 🐛 Cloudflare Tunnel Fix — API Server Middleware (05:52-05:55 UTC)
- **Problem:** Caddy's `handle /n8n/*` worked via direct VPS IP (`http://5.223.81.16/n8n/` ✅) but NOT via Cloudflare tunnel (`https://ps-vibe.com/n8n/` ❌ 404)
- **Root cause:** Cloudflare tunnel runs with `--token` (remote-managed), circumventing local Caddy. Tunnel routes directly to API server (port 8000), bypassing Caddy entirely.
- **Fix:** Added `@app.middleware("http")` in app.py that intercepts `/n8n/*` requests and proxies them via `httpx.AsyncClient` to `http://localhost:5678` (n8n container)
- Installed `httpx` (0.28.1) in the API server's venv at `/root/psvibe_api_server/venv/`
- **Issue #2:** n8n HTML at `/n8n/` loaded but blank.
  - **Root cause 1:** n8n's static asset paths are NOT prefixed with `/n8n/` (e.g., `/assets/index-xxx.js`, `/static/...`, `/favicon.ico`).
  - **Fix 1:** Updated middleware to also proxy `/assets/*`, `/static/*`, `/favicon.ico` to n8n.
  - **Root cause 2:** n8n frontend uses `window.BASE_PATH` (default `'/'`) for API calls. Since n8n's actual API is at `/n8n/rest/...`, the frontend made requests to `/rest/...` which 404'd.
  - **Fix 2:** Middleware intercepts `/static/base-path.js` and returns `window.BASE_PATH = '/n8n/'` instead of forwarding to n8n's default `'/'`.
- **Fix 3:** Added `n8n_assets_middleware` to handle rewritten `/n8n/assets/*`, `/n8n/static/*`, `/n8n/favicon.ico` paths by stripping `/n8n` prefix and proxying to n8n.
- **Fix bugs along the way**:
  - Missing `@app.middleware("http")` decorator on `n8n_assets_middleware`
  - Indentation errors in middleware body (wrong spacing after text replacement)
  - BASE_PATH replace pattern had wrong syntax (`'/,'` instead of `'/';`)
- **LESSON**: Never do complex multi-file edits via string replacement. Syntax errors pile up.
- **Final Result:**
  - `https://ps-vibe.com/` → Dashboard ✅
  - `https://ps-vibe.com/n8n/` → n8n Editor ✅ (fully rendered)
  - `https://ps-vibe.com/api/` → API Server ✅

FONT = "/tmp/NotoSansMyanmar_Google-Regular.ttf"

def bmp2pil(bm):
    raw = bytes(bm.buffer) if isinstance(bm.buffer, list) else bm.buffer
    w, h = bm.width, bm.rows
    if w==0 or h==0: return None
    if bm.pixel_mode == freetype.FT_PIXEL_MODE_MONO:
        d = bytearray()
        for r in range(h):
            for c in range(w):
                bi = r*bm.pitch + c//8
                bit = 7-(c%8)
                if bi < len(raw): d.append(0 if (raw[bi]>>bit)&1 else 255)
                else: d.append(255)
        return Image.frombytes('L', (w,h), bytes(d))
    else:
        return Image.frombytes('L', (w,h), bytes(255-b for b in raw))
