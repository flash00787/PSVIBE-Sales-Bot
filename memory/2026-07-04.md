### Bug Fix: Logout Glitch 🐛→✅
**Time:** ~13:10 UTC (19:40 MMT)

**Symptom:** Logout button caused UI glitch/flash — page didn't cleanly redirect to login.

**Root Cause:** Race condition between:
1. `auth.logout()` clearing token + `router.push("/login")` 
2. Mounted pages still making API calls → 401 responses → interceptor fired `window.location` redirect mid-navigation → confusion

**Fix:**
- **Interceptor (`api/index.js`):** Skip 401 redirect if already on login page (`!pathname.includes('/login')`)
- **Sidebar logout:** Changed `router.push("/login")` to `window.location.href = "/akt-clothing-shop/login"` — clean break, no API calls after logout

**Files Modified:**
- `frontend/src/api/index.js` — conditional 401 redirect
- `frontend/src/components/Sidebar.vue` — logout uses window.location
