## ЁЯза Key Lessons
- GROUP BY collapses pipe-delimited data тЖТ iterate all rows
- Elif chains must cover ALL variants (`"wave"` тЙа `"wavepay"`)
- systemctl restart can silently fail тЖТ verify PID, fallback `kill -9`
- `x if x else default` breaks on `0` тЖТ use `x if x is not None else default`
- PyMySQL `%` in LIKE тЖТ `CONCAT('%', '/', '%')` or `%%`
- FIFO for wallet: oldest topups consumed first; bonus min = 0 Ks
- Always use `per_price*qty` not raw `amount` for assets
